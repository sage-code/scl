# DBA Topic Description
Phase 1: Fundamentals
01 Database Systems Core database concepts, engine types, and application architecture.
02 Database Design Modeling, normalization, relation patterns, and design documentation.
03 Data Modeling Entity relationships, constraints, domains, and schema strategy.
04 SQL Fundamentals DDL, DML, query writing, and practical data manipulation.
Phase 2: Performance & Reliability
05 Indexing B-tree and hash indexing, selectivity, and index maintenance.
06 Query Optimization Execution plans, bottleneck diagnosis, and tuning workflow.
07 Transactions & Concurrency ACID, isolation levels, locking, and conflict handling.
Phase 3: Operations & Security
08 Security & Access Control Authentication, authorization, auditing, and secrets handling.
09 Backup & Recovery Backup strategies, restore testing, and disaster recovery plans.
10 Replication & High Availability Replication models, failover patterns, and resilient deployments.
11 Monitoring & Operations SLOs, alerting, capacity planning, and runbook-driven operations.
